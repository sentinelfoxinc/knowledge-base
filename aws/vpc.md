# AWS VPC Documentation

## Overview
Virtual Private Cloud (VPC) is a virtual network dedicated to your AWS account.

## Key Concepts
- Subnets
- Route Tables
- Security Groups
- Network ACLs

## Best Practices
- Use multiple AZs for high availability
- Implement proper security groups
- Use VPC flow logs for monitoring
